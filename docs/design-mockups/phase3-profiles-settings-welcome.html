D. Summary of Changes

Profiles View

  • 3-column card grid (icon-in-colored-circle, no stripe)
  • Stats grid: Tweaks, Gain, Risk, Restart per card
  • Thin risk bar (color matches risk level)
  • Apply button color matches profile risk (green/yellow/red)
  • Preview button per card
  • Compare panel (collapsible) for side-by-side stats
  • Fix hardcoded "Apply" in no-warnings branch

Settings View

  • Section headers: icon + uppercase title (palette, translate, shield, info)
  • Row layout: label + description on left, control on right
  • Toggle switches instead of checkboxes
  • New "Flight Recorder" toggle (already exists in batch scripts)
  • About section with app icon, version, description, GitHub/Docs links
  • Complete TH localization for all section headers and descriptions
  • Toast messages moved to UI_STRINGS

Welcome Overlay

  • Updated nav references: Dashboard→Home, Optimization→Tweaks/Profiles, Help→Docs
  • Replaced Unicode arrows (← →) in buttons with Material Symbols icons
  • Step icons use Material Symbols in accent-colored circles
  • Progress shown as dot indicators (not "1/5" text)
  • Skip button in top-right corner
  • Highlight boxes for key info per step
  • Step 5 uses success color (green) for "Ready" state
  • Gamer-friendly wording throughout

Cross-cutting

  • Zero emoji in all 3 views
  • All icons are Material Symbols Outlined (via icon_provider.py / icons.py)
  • All user-facing text in UI_STRINGS (EN + TH)
  • Colors from theme.py Sun Valley tokens only
