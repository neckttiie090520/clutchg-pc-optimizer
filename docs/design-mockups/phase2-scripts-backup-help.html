USB Selective Suspend
Power · Registry tweak
What it does
Prevents Windows from powering down USB ports to save energy. When this kicks in during gameplay, your mouse or keyboard can stutter for a few ms while the port wakes back up. Disabling it keeps ports always-on.
Impact
Risk LOW RISK
Effect Lower input latency
Trade-off Slightly higher idle power draw
Reversal

rocket_launch Getting Started
What is ClutchG?
A Windows optimizer for gamers. Applies reversible tweaks to cut latency, smooth out frame times, and kill unnecessary background noise — without touching Defender, UAC, or Windows Update.
Quick Start
1. Go to Tweaks → Profiles and pick one (Safe is a good start).
2. ClutchG auto-creates a backup before applying.
3. Restart if prompted.
4. Check Home for your updated score.
Key Points
• Everything is logged and reversible via Backup.
• Defender, UAC, and Windows Update are never touched.
• Risk levels: LOW = safe for everyone · MEDIUM = read first · HIGH = advanced only.

D3. Safety topic — Myth cards

Common Myths
MYTH: "Disabling Defender improves gaming performance"
FACT: Defender's real-world FPS impact is under 1%. Disabling it opens you up to malware. ClutchG never touches it.
MYTH: "More RAM = faster games"
FACT: RAM speed matters more than capacity above 16 GB. ClutchG tunes timing settings where safe.
MYTH: "BCDEdit /set useplatformtick yes always helps"
FACT: Helps some Intel CPUs, can hurt AMD Ryzen. ClutchG checks your CPU vendor first.
